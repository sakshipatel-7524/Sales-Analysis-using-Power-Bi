# Superstore Sales Analysis Dashboard

## Overview
This project presents a comprehensive **Superstore Sales Analysis Dashboard** built using **Power BI**. The dashboard provides an intuitive and visually appealing analysis of sales, profit, and other key metrics to derive actionable insights for business decision-making.

## Key Features
- **Interactive Regional Filters**: Analyze sales data across different regions (Central, East, South, West).
- **High-Level KPIs**:
  - Total Sales: 341.01K
  - Total Profit: 27.45K
  - Quantity Sold: 5239
  - Average Delivery Days: 4
- **Visualizations**:
  - Pie Charts for **Sales by Segment** and **Sales by Payment Mode**
  - Line Charts for **Monthly Sales Trends by Year (YoY)**
  - Bar Charts for **Sales by Ship Mode**, **Sales by Sub-Category**, and **Sales by Category**
  - Map Visualization for **Sum of Sales and Profit by State**

## Sales Forecasting
Additionally, the project includes a **15-day sales forecast** to help stakeholders plan inventory, marketing, and logistics efficiently.

## Insights Derived
1. **Sales Distribution**:
   - Consumer Segment accounts for the highest sales (48%).
   - Online Payment Mode contributes 39% to total sales.
2. **Shipping Preferences**:
   - Standard Class is the most preferred shipping mode (78K sales).
3. **Category Performance**:
   - Office Supplies leads in sales (0.15M), followed by Furniture (0.11M) and Technology (0.09M).
4. **Regional Performance**:
   - Sales and profit distribution varies significantly across states, as shown on the map visualization.
5. **Year-on-Year Trends**:
   - Sales demonstrate a consistent increase towards the end of the year.

## Tools Used
- **Power BI**: For data visualization and dashboard creation.
- **Forecasting**: Built-in forecasting tools in Power BI for projecting 15-day sales trends.

## How to Use
1. Download the `.pbix` file from the repository.
2. Open it in Power BI Desktop to explore the dashboard interactively.
3. Adjust filters to analyze specific regions, categories, or time periods.
